2. button标签

在 button 元素内部，您可以放置内容，比如文本或图像。这是该元素与使用 input 元素创建的按钮之间的不同之处。

相比，提供了更为强大的功能和更丰富的内容。 标签之间的所有内容都是按钮的内容，其中包括任何可接受的正文内容，比如文本或多媒体内容。

语法
值 描述
reset 重置按钮，清除表单数据
button 可点击按钮，IE的默认值
submit 提交按钮，是默认值（除了IE）
注意：请始终为按钮规定 type 属性
