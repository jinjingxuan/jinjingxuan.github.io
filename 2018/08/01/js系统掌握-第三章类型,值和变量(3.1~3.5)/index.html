第三章类型,值和变量

3.1数字

和C，Java不同，JS不区分整数值和浮点数值。JS所有数字均用浮点数值表示。

当一个数字直接出现在JS程序中，我们称之为数字直接量。

3.1.1整形直接量

十进制整数：0 ，3

十六进制：以 “0x” 或 ‘’0X‘’ 为前缀，如 0xff ，0xCAFE911。

八进制：有些支持，有些不支持，尽量不使用

3.1.2浮点型直接量

例如：

​ 3.14，.333，6.02e23，1.24523E-32

3.1.3JS中的算术运算

  1. 本身算术运算符
  2. Math对象的属性定义的函数,如Math.pow(2,53)

js中的算术运算在溢出(overflow)，下溢(underflow)或被零整除时不会报错。

溢出：数字运算结果超出了js所能表示的数字上限，结果为infinity。

下溢：当运算结果无限接近于0并比js能表示的最小值还小的时候发生的情形，js会返回0.

​ 例如：-1 / Infinity 返回负零

被零整除：返回NaN。

​ 但是有一个例外，0/0是没有意义的，返回NaN(not-a-number)

NaN有一点特殊，它和任何值都不相等，包括自身，那么如何判断x是否为NaN？

  1. 不能用 X==NaN判断
  2. 使用x!=x判断，当且仅当x为NaN的时候，表达式结果为true
  3. isNaN() 参数是NaN或者一个非数字值，则返回true
  4. 类似的 isFinite() ,参数不是NaN,Infinity的时候返回true

3.1.4 二进制浮点数和四舍五入错误

实数有无数个，但JS通过浮点数的形式只能表示有限的个数。

JS的数字具有足够的精度，可以及其近似于0.1,但是还是会存在问题。

这个计算结果可以胜任大多数的计算任务，上述问题也只有在比较两个值是否相等的时候才会出现。

3.1.5 日期和时间

Date() 构造函数，用来表示日期和时间的对象。这些日期对象的方法为日期计算提供了简单的API。

3.2文本

JS定义的各式字符串操作方法均作用于16位值，而非字符。

3.2.1 字符串直接量

字符串直接量是由单引号或双引号括起来的字符序列。

ECMAScript 3中，字符串直接量必须写在一行中

ECMAScript 5中，字符串可以拆分成数行，每行必须以反斜杠（\）结束

JS和Html代码混杂在一起的时候，最好有各自的引号风格

3.2.2 转义字符

JS字符串中，\有着特殊的用途，\加上一个字符就是一个转义字符。

3.2.3字符串的使用

JS的内置功能之一就是字符串连接。

要确定一个字符串的长度，可以使用length属性

除了length属性，字符串还提供许多可以调用的方法（先放这些）

函数 功能
substr(start,length) 表示从start位置开始，截取length长度的字符串。
substring(start,end) 从start到end之间的字符串，包括start位置的字符但是不包括end位置的字符。
split() 使用一个指定的分隔符把一个字符串分割存储到数组
indexOf() 返回字符串中匹配子串的第一个字符的下标
charAt() 选中当前位置的字符

ES5中，字符串可以当作只读数组。

3.4null和undefined

null常用来描述空值，对null执行typeof预算，结果返回字符串object，可以将null认为是一个特殊的对象值。

undefined，未定义的值表示更深层次的“空值”，表明变量没有初始化。，typeof返回undefined

3.5全局对象（gloabl object）

全局对象的属性是全局定义的符号，JS程序可以直接使用。当JS解释器启动时（或者任何Web浏览器加载页面的时候），他将创建一个新的全局对象，并给他一组定义的初始属性。

全局 举例

全局对象 Math和JSON

在代码的最顶级，不在任何函数内的JS代码，可以使用关键字this引用全局对象
