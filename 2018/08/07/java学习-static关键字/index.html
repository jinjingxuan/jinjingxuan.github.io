java学习-static关键字

发表于 2018-08-07 | 分类于 java学习
字数统计: 361 | 阅读时长 ≈ 1

1. static定义属性

一个类的主要组成就是属性和方法，而每个对象都分别拥有各自的属性内容，如果类中的某个属性希望定义为公共属性（所有对象都可以使用的属性），则可以在声明属性前加上static关键字。

java中常用内存区域

内存区域 作用
栈内存区域 保存所有的对象名称
堆内存区域 保存每个对象的具体属性内容
全局数据区 保存static类型的属性
全局代码区 保存所有的方法定义

既然static是一个公共属性的概念，那么如果只是简单地由一个对象去修改static属性的做法很不合适，最好的做法是由所有对象的公共代表来进行访问。

与非static属性的区别

非static属性必须产生实例化对象才可以访问。

static属性不受实例化对象的控制，在没有实例化对象产生的情况，依然可以使用static属性。

2. static定义方法

使用static定义的方法也可以在没有实例化对象产生的情况下由类名称直接调用。

  • static方法不能直接访问非static属性或方法，只能调用static属性或方法。
  • 非static方法可以访问static的属性或方法，不受任何的限制。
