组件传值

前言：在写vue代码时,npm run dev总会报一些语法错误，但是在文件中看不到，所以在下载插件ESLint后解决。下载后发现ESLint对代码规范要求很高，比如缩进为2个空格键，冒号后要有空格，虽然很麻烦，我觉得刚学习还是按照规范来比较好，我就在setting中设置了tab键为两个空格。

父子组件的关系可以总结为 prop 向下传递，事件向上传递。父组件通过 prop 给子组件下发数据，子组件通过事件给父组件发送消息

父组件传子组件：通过props

子组件传父组件：$emit

  • 输入框的onchange事件，要在 input 失去焦点的时候才会触发；
  • oninput 事件在用户输入时触发，它是在元素值发生变化时立即触发；

props与data的区别

  • data是每个组件的私有内存，可以在其中存储需要的任何变量。
  • props是将数据从父组件传递到子组件的方式。

都是响应式的。
