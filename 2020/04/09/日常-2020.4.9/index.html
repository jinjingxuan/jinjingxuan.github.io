1.什么是npm

NPM 的思路大概是这样的：

这些可以被使用的代码被叫做「包」（package），这就是 NPM 名字的由来：Node Package Manager

可以安装淘宝镜像cnpm代替提高速度

npm install -g cnpm --registry=https://registry.npm.taobao.org

然后安装webpack

cnpm install webpack -g

然后输入webpack -v会提示安装webpack-cli

cnpm install webpack-cli -g
