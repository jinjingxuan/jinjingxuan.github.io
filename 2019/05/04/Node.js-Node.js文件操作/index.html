Node.js的文件操作API由fs（File System）模块提供，该模块提供的函数均有异步和同步版本，例如读取文件内容的函数有异步的fs.readFile()和同步的fs.readFileSync()。实际开发中建议大家使用异步函数，性能更高，速度更快，而且没有阻塞。
